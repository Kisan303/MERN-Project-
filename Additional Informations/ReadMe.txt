How to Test the Endpoint

BACKEND INFO

Step 1: Open Terminal
Navigate to the backend directory:
cd backend

Step 2: Start the Backend Server
Run the following command to start the backend server:
npm start

Backend URLs:

- If you want to fetch all user details:
  http://localhost:5000/api/users

- If you want to fetch details of a specific user by their ID (Example: "_id": "6747bfae79c6ed9aaed0e605"):
  http://localhost:5000/api/users/674900dd0f375e1fbe7aea90

FRONTEND INFO

Step 1: Open Terminal
Navigate to the frontend directory:
cd frontend

Step 2: Start the Frontend Server
Run the following command to start the frontend server:
npm start

Frontend URLs:

- Default Login Page:
  http://localhost:3000/
